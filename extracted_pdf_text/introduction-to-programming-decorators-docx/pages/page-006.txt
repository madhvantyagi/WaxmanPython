When decorators are applied, they must appear on their own line immediately prior to the
function.
Now that we have the decorator we can apply it to any function we like. Say you have a function
def mult(x,y):
return x*y
I can decorate it like this:
import functools
def my_decorator(func):
@functools.wraps(func)
def wrapper(*args, **kwargs):
""" wrapper"""
print("Before the function is called.")
result = func(*args, **kwargs)
print("After the function is called.")
return result
return wrapper
@my_decorator
def mult(x,y):
""" mult func"""
return x*y
z=mult(2,3)
print(z,mult.__name__) # Output: "my_function"
print(z,mult.__doc__)
and I get
Before the function is called.
After the function is called.
